# Top-rated-movies-Analysis
Project Overview
This project focuses on analyzing and visualizing top-rated movies data using MySQL for data storage and querying, and Power BI for interactive dashboards.
The goal is to extract meaningful insights such as movie ratings trends, popular genres, and audience preferences, and present them in an easy-to-understand visual format.
